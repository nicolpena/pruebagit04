Tipos de datos primitivos en java

Tipo int

Es un tio de datos numérico de 32 bits, enetero con signo.
Su rango va desde -2e31 hasta 2e31 - 1.

Tipo float

El float es un tipo de dato para almacenar números decimales con precisión de 64 bits.
